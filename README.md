# Flashcard Quiz 
An application that allows the user to test themselves on topics of their choosing with a .txt file

## Instructions
1. Download this GitHub Repository
2. Run the code (preferablly with Visual Studio for Winform Integration)
3. Select a file of your choosing with the following constraints
   - It must have 5 parts: Question text and 4 answers separated by the delimeter |
   - One answer must be marked with an asterisk (*)
   - It must not be blank
4. Test Yourself!
